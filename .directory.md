# Корень проекта MTOT

## Назначение
Корневая директория проекта More Than One Touch — клиентского мода для Minecraft, реализующего систему многоклавишных комбинаций.

## Ключевые файлы
- `PROJECT_MAP.md` — карта взаимосвязей проекта
- `MTOT-doc.md` — техническая спецификация
- `2R-skill-2.0.md` — стандарт разработки

## Связи
- `common/` — общий код
- `fabric/` — Fabric обёртка
- `neoforge/` — NeoForge обёртка
- `forge/` — Forge обёртка
