# Папка `forge/`

## Назначение
Forge-специфичная обёртка — инициализация клиента и управление клавишами через Forge events (Minecraft 1.20.1 LTS).

## Ключевые файлы
- `MTOTForgeClient.java` — точка входа Forge
- `ForgeKeyManager.java` — управление клавишами через Forge

## Связи
- Вход: из Forge (события)
- Выход: в `common/core/` (tick, register)
