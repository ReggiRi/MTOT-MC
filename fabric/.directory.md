# Папка `fabric/`

## Назначение
Fabric-специфичная обёртка — инициализация клиента и управление клавишами через Fabric API.

## Ключевые файлы
- `MTOTFabricClient.java` — точка входа Fabric
- `FabricKeyManager.java` — управление клавишами через Fabric

## Связи
- Вход: из Fabric API (события)
- Выход: в `common/core/` (tick, register)
