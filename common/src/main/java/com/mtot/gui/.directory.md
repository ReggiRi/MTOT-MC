# Папка `gui/`

## Назначение
Пользовательский интерфейс — экран настроек и виджеты для управления привязками.

## Ключевые файлы
- `MTOTControlsScreen.java` — экран настроек управления
- `BindingEntryWidget.java` — виджет строки привязки

## Связи
- Вход: из `core/` (список привязок)
- Выход: в `core/` (изменение привязок)
