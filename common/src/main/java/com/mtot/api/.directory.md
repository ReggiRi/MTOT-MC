# Папка `api/`

## Назначение
Публичное API мода — интерфейсы и модели, доступные другим модам для интеграции.

## Ключевые файлы
- `IMTOTAPI.java` — главный интерфейс API
- `KeyCombination.java` — модель комбинации клавиш
- `Modifier.java` — enum модификаторов (SHIFT, CONTROL, ALT)

## Связи
- Выход: в `core/` (реализация)
- Вход: от других модов (интеграция)
