# Папка `core/`

## Назначение
Ядро системы — регистрация комбинаций, отслеживание состояний клавиш, управление привязками.

## Ключевые файлы
- `MTOTManager.java` — синглтон, управляющий всеми комбинациями
- `BindingRegistry.java` — реестр действий и привязок
- `KeyStateTracker.java` — отслеживание нажатий клавиш

## Связи
- Вход: из `api/` (регистрация), из `gui/` (изменения)
- Выход: в `config/` (сохранение), в платформенные обёртки (выполнение)
