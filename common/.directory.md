# Папка `common/`

## Назначение
Общий код, независимый от платформы — 95% логики мода.

## Ключевые файлы
- `src/main/java/com/mtot/api/` — публичное API
- `src/main/java/com/mtot/core/` — ядро системы
- `src/main/java/com/mtot/gui/` — пользовательский интерфейс
- `src/main/java/com/mtot/config/` — конфигурация
- `src/main/java/com/mtot/actions/` — встроенные действия

## Связи
- Вход: из платформенных обёрток (tick, register)
- Выход: в платформенные обёртки (через API)
