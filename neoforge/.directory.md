# Папка `neoforge/`

## Назначение
NeoForge-специфичная обёртка — инициализация клиента и управление клавишами через NeoForge events.

## Ключевые файлы
- `MTOTNeoForgeClient.java` — точка входа NeoForge
- `NeoForgeKeyManager.java` — управление клавишами через NeoForge

## Связи
- Вход: из NeoForge (события)
- Выход: в `common/core/` (tick, register)
